# naughty-correct

Warning: Mild profanity.

There are days when I almost lose my shit typing on my phone.
Autocorrect fails to correct non-existent words, autocorrects
correct words, and ignores repeated attempts to actually write
anything at all.

So I thought I'd write a mini program that irritates the user, 
just to see if I can get inside the mind of autocorrect.

For the avoidance of doubt, please note well that the next sentence
I type is in jest.

Maybe I could deploy this as WhatsApp malware!

Update: I've now figured out how to add multiple words to my
naughty dictionary! Hashes, of course. This is exciting.
